Detailed Description

ZeroMQ error.

Exception to identify ZeroMQ errors. The constructor has no arguments as ZeroMQ handles errors using a global variable which is evaluated using zmq_errno() inside the exception.

try
{
// code with zeromq
}
catch (zmq::error &e)
{
// handle exception
}
