7 ## Problem
8 
9 We can use the `void_t` trick shown in the above video to detect
10 whether a type has a member function, overloaded operator, etc. In
11 this example we want to detect whether a type has a subscript operator
12 with subscript `int`.
13 
14 ## Solution
15 
16 The `void_t` uses expression SFINAE to detect the existence of
17 something. The code is ultra short.
18 ```cpp
19 template < typename T, typename Index >
20 using subscript_t = decltype(std::declval<T>()[std::declval<Index>()]);
21 
22 template < typename, typename Index = int, typename = void_t<> >
23 struct has_subscript : std::false_type {};
24 
25 template < typename T, typename Index >
26 struct has_subscript< T, Index, void_t< subscript_t<T,Index> > > : std::true_type {};
27 ```
28 We could make it even shorter by omitting `subscript_t`. For
29 explanation watch the video.
